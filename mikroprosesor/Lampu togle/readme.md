Lampu Toggle
Membuat lampu toggle pada ATmega328 / Arduino dengan model:

1. sampling dengan loop
2. sampling dengan delay
3. sampling dengan interupsi timer
4. wait loop
5. interupsi external (INTx / PCINTx)

Pengembangan menggunakan:
Arduino IDE
proteus
